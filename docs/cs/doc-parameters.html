Parametry v šablonách

HREF

Vytvoří odkaz na daný controller a metodu. Controller musí být celé jméno třídy. Metodou se myslí jméno funkce. Controller a metoda jsou odděleny :. Parametry jsou za jménem metody a jsou opět odděleny :. Pokud parametr neobsahuje =, je použit jako součást URI, Jinak je to GET parametr. 


<a t:href="" />
<!-- example - approximate result: /modulename/my-controler/action -->
<a t:href="org.example.app.controllers.MyController:myAction" />
<!-- example with URL parameter - approximate result: /modulename/my-controler/action/123 -->
<a t:href="org.example.app.controllers.MyController:myAction:123" />
<!-- example with more URL parameters - approximate result: /modulename/my-controler/action/123/a -->
<a t:href="org.example.app.controllers.MyController:myAction:123:a" />
<!-- example with URL parameter and GET parameter - approximate result: /modulename/my-controler/action/123?foo=bar-->
<a t:href="org.example.app.controllers.MyController:myAction:123:foo=bar" />

Pro vytvoření odkazu mimo aplikaci, použijte prefix A. 


<a t:href="A:https://example.com:org.example.app.controllers.MyController:myAction" />

Title

Vypíše přeložený text jako parametr title. 


<div t:title="" />
<!-- example -->
<div t:title="messages.tooltips.div-title" />

Placeholder

Vypíše přeložený text jako parametr placeholder. 


<input t:placeholder="" />
<!-- example -->
<input t:placeholder="messages.placeholders.input-help" />
