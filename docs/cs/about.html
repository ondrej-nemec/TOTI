O TOTI

TOTI - Tip Of The Iceberg (Špička ledovce) - je framework psaný v Javě. Poskytuje HTTP(S) rozhraní mezi aplikací a uživately. Umožňuje přidat aplikaci Rest API, generovat HTML ze šablon nebo jen spuštět webový server (jako Apache ale jednodušší - rozumněj primitivnější).

Hlavní účel TOTI je vytvoření GUI pro java aplikace, které běží jako server, například v Industry 4.0 (IIoT, Průmyslový internet věcí) or IoT (Internet věcí). TOTI je snadné včlenit do desktop aplikací. Může být použito jen jako MVC framework (jako třeba Symfony nebo Nette v PHP nebo ASP.NET v C#), ale to není hlavním učelem.

Výhody

  • Snadné začlenění
  • Není potřeba instalovat server
  • Parsování šablon do HTML a ne do servletů
  • Může být použito v serveru nebo desktopové aplikaci

Dependency

Jaké jsou možnosti přidání TOTI do vaší aplikace, najdete na JitPack nebo rovnou použijte Gradle:

gradle.build 

repositories { 
	...
	maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.ondrej-nemec.TOTI:Tag'
}

Pokud chcete použít šablonovací system v produkci, je nutné přidat tools.jar do buildpath. V Javě 8 se to dělá: 


implementation files("${System.properties['java.home']}/../lib/tools.jar")
