Template parameters

HREF

Create link to given controller and method. Controller must be full class name. Method is name of function. Controller and method are splited by :. 


<a t:href="" />
<!-- example -->
<a t:href="org.example.app.controllers.MyController:myAction" />

Title

Print translated message as title parameter. 


<div t:title="" />
<!-- example -->
<div t:title="messages.tooltips.div-title" />
