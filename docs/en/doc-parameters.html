Template parameters

HREF

Create link to given controller and method. Controller must be full class name. Method is name of function. Controller and method are splited by :. URL parameters are after method name and splited by :. 


<a t:href="" />
<!-- example -->
<a t:href="org.example.app.controllers.MyController:myAction" />
<!-- example with parameter -->
<a t:href="org.example.app.controllers.MyController:myAction:123" />

Title

Print translated message as title parameter. 


<div t:title="" />
<!-- example -->
<div t:title="messages.tooltips.div-title" />

Placeholder

Print translated message as input placeholder. 


<input t:placeholder="" />
<!-- example -->
<input t:placeholder="messages.placeholders.input-help" />
