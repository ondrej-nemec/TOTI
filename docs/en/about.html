About TOTI

TOTI - Tip Of The Iceberg - is Java web framework. Provide HTTP(S) interface between users and your Java application. With TOTI you can create Rest API for your app, generate HTML pages from templates or just run web server (like f.e. Apache but begging).

Main purpose of TOTI is create framework for GUI for Java servers f.e. in Industry 4.0 (IIoT) or IoT. The TOTI is prepared for easy embedded into desktop applications. It can be used as MVC framework (like Symfony or Nette in PHP, or ASP.NET in C#) but this is not main purpose.

Benefits

  • Easy include
  • No server instalation
  • Parsing templates to HTML not to servlets
  • Can be used as server or in desktop applications

Dependency

How to add TOTI to your project project see JitPack or you can use Gradle:

gradle.build 

repositories { 
	...
	maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.ondrej-nemec.TOTI:Tag'
}

If you wish use templating system in production, you need add tools.jar to buildpath. In Java 8 it be like: 


implementation files("${System.properties['java.home']}/../lib/tools.jar")
